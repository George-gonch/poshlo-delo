# Сайт «Пошло дело»

Это простой статический сайт: HTML + CSS + JS. Его можно открыть на компьютере без сервера и бесплатно выложить на GitHub Pages, Netlify или Vercel.

## Как открыть сайт на компьютере

1. Распакуйте архив.
2. Откройте файл `index.html` двойным кликом.
3. Сайт откроется в браузере.

## Как выложить бесплатно через GitHub Pages

1. Зарегистрируйтесь на https://github.com
2. Создайте новый репозиторий, например `poshlo-delo`.
3. Загрузите в него файлы `index.html`, `styles.css`, `script.js`.
4. Откройте Settings → Pages.
5. В разделе Build and deployment выберите Source: Deploy from a branch.
6. Branch: `main`, folder: `/root`.
7. Нажмите Save.
8. Через несколько минут сайт будет доступен по адресу вида `https://ваш-логин.github.io/poshlo-delo/`.
